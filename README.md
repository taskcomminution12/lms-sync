# lms-sync

## 주요 기능

### 1. 대시보드 & 과목 페이지

![대시보드](https://raw.githubusercontent.com/taskcomminution12/lms-sync/main/main_view.png)

**대시보드**
- 메인 UI : 수강 과목 수, 저장된 자료 수, 임박 과제 수
- 차트: 주차별 자료 누적 현황, 과제 마감 분포 도넛 차트
- 임박 과제 테이블 (D-day 뱃지 표시)
- 활동 피드: 최근 다운로드 내역
- 캘린더: 과제 마감일 자동 표시

**과목 페이지**
- 주차별 자료 목록
- 게시판 / 공지 / 과제
- 드래그 & 드롭 영역

---

### 2. 자료 자동 다운로드

![자료 다운로드](https://raw.githubusercontent.com/taskcomminution12/lms-sync/main/download_file.png)

- 수동 동기화, 자동 동기화 시점마다 lms의 파일 다운로드
- 각 과목들은 /폴더/과목_분반/주차/다운로드 파일 형태로 구조화

---

### 3. 과목 페이지 드래그 & 드롭

![드래그 & 드롭](https://raw.githubusercontent.com/taskcomminution12/lms-sync/main/file_upload.png)

- 과목 페이지에서 파일을 프로그램 창 위로 드래그
- 주차 헤더 위에 놓으면 해당 주차 폴더 하위에 저장
- 빈 공간에 놓으면 `_수동자료/` 폴더에 저장
- 자동으로 DB 인덱싱 및 페이지가 갱신

## 사용 방법

### 초기 설정 (필수)

1. **토큰 입력**
   - 프로그램 실행 후 **설정 → 토큰 입력** 으로 이동
   - 아래 방법으로 토큰 발급 후 붙여넣기

   **토큰 발급 방법**
   1. LMS 로그인
   2. **계정 → 설정 → 승인된 통합으로 이동**
   3. **새 액세스 토큰 만들기** 클릭
   4. 생성된 토큰 복사 후 프로그램에 붙여넣기

2. **다운로드 폴더 설정**
   - **설정** 에서 파일이 저장될 폴더 경로 지정

3. **동기화 시작**
   - **대시보드 → 메인 동기화** 클릭

---

### 실행
다운로드한 `.exe` 파일을 실행하면 백그라운드에서 동작합니다.

### 종료 방법 (Windows)
1. 작업 표시줄 오른쪽 하단의 **숨겨진 아이콘 표시** (▲) 클릭
2. 파란색 **SU** 아이콘을 **우클릭**
3. 종료 선택

---

### 참고 사항

- **카카오톡 연동**: REST API Key가 필요하며 데모 버전에서는 동작하지 않습니다.
- **자동 동기화 주기**: 데모 버전이며 30분으로 고정하는것을 권장합니다.
